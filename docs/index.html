About

CST is a simple coming soon template that provides a beautiful view for any kind of projects.

It has a lot of nice features with the existing HTML and CSS, making it a great choice for dynamic and responsive layouts.

How to install

CST contains 4 standard buttons for navigation, every button opening a new page in a full screen popup. Also has support for MailChimp subscription.

1. Upload cst folder to your server. This folder contains all of the asssets of the theme, folders and files.

  • css
  • images
  • js
  • lib
  • svg

2. The following files in your <head> section of the page can be replaced for new colors. Please ensure that all of them have the correct path: 


<!-- include css files -->
<link rel="stylesheet" href="css/main.css">
<link rel="stylesheet" href="css/animate.css">
<link rel="stylesheet" href="css/style-light.css">

Be aware that if you want only one color for your template leave a single css file in <head>. Do not remove animate.css from the head because it's used for some animations.


3. Javascript files can be found in the footer of the theme before the closing tag </body>.

HTML

The HTML is devided in:

  • Home
  • About Us
  • Subscribe
  • Services
  • Contact Us

    • Home

    This section is the main page of template, every link has a corresponding id #about-12, #services-12, #subcribe-12 and #contact-12. If you change the id from attribute data-model then you must change the id from the div section.

    
<ul class="navbar-collapse collapse">

    <li><a class="btn-1f md-trigger" href="#about" data-model="about-12">About Us</a></li>

    <li><a class="btn-1f md-trigger" href="#services" data-model="services-12">Services</a></li>

    <li><a class="btn-1f md-trigger" href="#subscribe" data-model="subscribe-12">Subscribe</a></li>

    <li><a class="btn-1f md-trigger" href="#contact" data-model="contact-12">Contact Us</a></li>

</ul>

<div class="modall...overlay-slidedown" id="about-12">

    From the main you can change timer date and introduction text. For example, intro text can be found in the fallowing html tags: 

        <a href="#" class="logo"></a>

    The image for logo can be found in images folder with the name logo.png, replace this image with your custom logo. Also the date from the timer can be changed from main.js file.

        $('#timer').downCount({
        date: '09/09/2016 12:00:00',
        offset: +2
    });

    About Us

    About Us section contains a simple carousel and description about the project you want to open. This is very simple popup and begin from the fallowing tag.

        <!-- About Us Modal md-effect-12 -->
    <div class="modall custom-modal md-modal about-12 overlay-slidedown" id="about-12">

    Subscribe

    Subscribe template has a simple input where a user must add the email address. This form is connected to MailChimp service. You must have an account on MailChimp create a group and copy the api key and list id that they provide you. Replace them in subscribe.php file and you are conected. For example

        $api_key = "ce871351b260893257cbe0a21f876db2-us11";
    $list_id = "e262fdad28";

    Subscribe template begins from the fallowing tag:

        <div class="modall custom-modal md-modal subscribe-12 overlay-slidedown" id="subscribe-12">

    Services

    This template begin from the fallowing tag

    <div class="modall custom-modal md-modal services-12 overlay-slidedown" id="services-12">

    You can replace the name of categories and add more description to your services.

    Contact Us

    Contact section has two parts, a zone where you show your address and social profiles and a Contact form. For the functionality of contact we used ajax and PHP, for how to change email addres see PHP section.

CSS

The theme is builded with SASS and every section from the html is devided in partials for future changes. Also we use Bootstrap as a ground css framework. All the assets of sass can be found in style folder. For more information of how to install sass, please see this link.

PHP

Contact form from HTML is using PHP for backend. The file is contact.php, here you must change the address from line 3 with the desire email address. $to_email = "test@email.com";, anything else must remain the same. You must also check your hosting to have a php version installed, if you don't have please contact your hosting support. If you need some custom modifications for php please send us an email at : support@spritemedia.net and we will get in touch soon as possible.

Javascript
